About

Anti-Adblock Killer Continued

AAK-Cont is a compatibility shim for uBlock Protector, if you are using Chromium / Chrome and uBlock Origin, please use uBlock Protector instead.

Please DO NOT install AAK-Cont if you are using uBlock Protector, it will cause undefined behavior.

Composed of a Userscript «AAK-Cont Script» written in JavaScript and a few filter lists «AAK-Cont filter lists» designed for the ad blocker you use. They are complementary and unlock different website.

You can report a issue or contribute to the project by visiting our «GitLab» repository.
